Please consider contributing to Gnofract 4D. If you aren't a programmer, you can still contribute by reporting bugs, writing formulas or coloring methods, creating gradients, or just by posting your images to the User Gallery to inspire others.

If you're familiar with Python or C++, there's plenty you could work on, regardless of your level of experience. Some example projects could be:

  • Adding arbitrary precision math support using GMP
  • Compiler improvements such as array support and optimizations
  • Distributed computation to speed up calculations
  • Improving the UI, such as making a better formula browser

See the TODO list for a longer list of possibilities. If you're interested please mail me.
