Download Information

Gnofract 4D works on modern Unix-based operating systems such as Linux and FreeBSD. It can also be run on Mac OS X. It is not currently available for Windows.

There are a number of different ways to install Gnofract 4D depending on your operating system and/or distribution. See for details.

Packages from elsewhere

Other people have kindly repackaged Gnofract 4D for additional Linux distributions and OSes. You may find these useful. However please note I don't maintain these and in most cases have never used them. They are often for an older release than the current one, especially if the new release has only just come out.

Other RPMs Dag Wieers has produced RPM packages for a number of distributions. Fedora 3 RPM Package
Fedora 4 RPM Package Fedora 5 RPM Package Fedora 6 RPM Package Gnofract 4D is available in Fedora Extras.
